Estrutura Básica

(voltar)

Para o html temos a seguinte estrutura básica.

  • <!DOCTYPE html>: Início do documento.
  • <html>: Raiz do HTML.
  • <head>: Informações da página.
  • <body>: Conteúdo visível.

Estilos com Tags HTML (Inline)

(voltar)

Essas tags afetam o estilo do texto diretamente:

  • Texto em negrito com <b>
  • Texto em itálico com <i>
  • Texto sublinhado com <u>
  • Texto marcado com <mark>
  • Texto pequeno com <small>
  • Texto deletado com <del>
  • Texto inserido com <ins>
  • Texto citado com <blockquote>
  • Texto subescrito com <sub>
  • Texto sobrescrito com <sup>
  • Passa o mouse <abbr>
